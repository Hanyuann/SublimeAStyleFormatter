SublimeAStyleFormatter
======================

SublimeAStyleFormatter is a simple code formatter plugin for Sublime Text 2.
It provides ability to format C, C++, C#, and Java files.

Prerequisites (Linux Only)
--------------------------
Because ctypes can't be imported in Linux right now in the Linux versions of Sublime Text 2.
Workaround is provided with the power of pythonbrew:

    SUBLIME_TEXT2_FOLDER="<your Sublime Text 2 folder>"
    curl -kL http://xrl.us/pythonbrewinstall | bash
    source "$HOME/.pythonbrew/etc/bashrc"
    pythonbrew install --configure="--enable-unicode=ucs4" 2.6
    ln -s $HOME/.pythonbrew/pythons/Python-2.6/lib/python2.6/ "${SUBLIME_TEXT2_FOLDER}/lib/python2.6"

**NOTE (Linux Only)**
Linux astyle binaries libastyle.so and libastyle64.so are built with Ubuntu 12.04,
if you have any problems with these binaries, you have to build with your own:

1. Install g++ and [scons](http://www.scons.org)
2. Clone astyle-build-scons using git:

        git clone git://github.com/timonwong/astyle-build-scons.git astyle-build
        cd astyle-build
        git submodule init
        git submodule update

3. Build with scons:

        scons --32  # Build i386 binary
        scons --64  # Build x86_64 binary


Usage
-----
### Key Bindings
The default key bindings for this plugin:

**Windows, Linux:**
+ `ctrl+alt+f`: Format current file
+ `ctrl+k, ctrl+f`: Format current selection

**OSX:**
+ `super+alt+f`: Format current file
+ `super+k,super+f`: Format current selection

### Command Palette
Open the command palette, it apperas as `SublimeAStyleFormatter: Format Current File`
and `SublimeAStyleFormatter Format Current Selection`.
